Scope

const amd let are blocked scope - their value belongs to the block they are declared in.

Global Scope

These are declared outside of a block. Avoid these because if they are global then they cannot be used again.

Local Scope

any variables defined within a block using const and let will only be used in that particular block.

Converting Strings to Numbers

Using the 'Number' method is the best way to change a string to a number - Number('23')

The string will not be able to be changed to a number with Number('hello'). If you try, it will result in a NaN.

Type coercion can also be used - const answer = '5' * 1 which will = 5 and be a 'number' type. Another way could be to add a + in front of it - const answer = +'5' which will, again, result in '5'.

They should not be used everytime because they can be hacky.

Converting Numbers to Strings

Using the String function is the prefered way of changing a number to a string - String(3) will result in '3'.

This can also be coerced by adding an empty string - 3 + '' This is not recommended

Another option is to use toString() - 10..toString() reults in '10'.

10..toString(10) changes to base number to binary '1010'

Parsing

parseInt() can be used to convert a string representation of a numerical value back into a number.

parseInt('1010', 2) converts from binary, back to decimal - 10

If a string starts with a number, the parseInt function will use the number and ignore the letters the come after:

const address = '221B Baker Street'; parseInt(address, 10) - 221 : This returns a NaN when checking if this is a number - Number(address)

If you use parseInt with a decimal it will remove anything after the decimal point:

parseInt('2.9',10) - 2

parseFloat will convert strings into floating point decimal numbers:

parseFloat('2.9',10) - 2.9

Template Literals

These are special types of string introduced in ES6 - they use the back tick character to deliminate the string

example - 'Hello!'

Both types of quote marks can be used within the string which is an advantage:

example - 'She said, "It's me!"'

Javascript expressions can be inserted inside a string and the result can be displayed:

const name = 'SIRI'; 'Hello ${ name }!'; - 'Hello SIRI!'

const age = 39; 'I will be ${ age + 1 } next year'; - 'I will be 40 next year'

They can also include line breaks using \n - each one for a new row.

Array Properties and Methods

example of an array:

const avengers = ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow'];

avenger.length; - counts how many items are in the array

avengers[avengers.length - 1]; - 'Black Widow' : this shows the last item in the list.

Arrays can be made longer than the list but the additional slots will be filled with 'undefined'

avengers.length = 8; avengers

result : ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow', undefined, undefined]

You can also make the array shorter - it will then just show the items you count:

avengers.length = 3; avengers

result : ['Captain America', 'Iron Man', 'Thor']

Pop, Push, Shift, and Unshift

To remove the last item from an array : avengers.pop()

To remove the first item from an array : avengers.shift()

To append a new item to the end of an array : avengers.push('Thor')

To apend a new item to the beginning of an array : avengers.unshift('Captain America')

Merging Arrays

concat() can be used to merge arrays with one or more arrays

avengers.concat(['Hulk','Hawkeye', 'Black Widow']);

result : ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

This does not change the original array - it creates another array.

An alternative to this is to use the new ES6 spread operator which is three dots, ...

This has the affect of spreading out all the values in each array which allows them to be placed inside a new array.

avengers = [ ...avengers, ...['Hulk','Hawkeye', 'Black Widow'] ];

Join() Method

This can be used to turn an array into a string that includes all the items which are separated by commas.

avengers.join()

result : 'Captain America, Iron Man, Thor, Hulk, Hawkeye, Black Widow'

You also have the option of choosing a different separator:

avengers.join(' & ')

result : 'Captain America & Iron Man & Thor & Hulk & Hawkeye & Black Widow'

Slicing and Splicing

Using slice() creates a subarray - it chops out a slice of the original array. It starts at the first position and finishes at another

avengers.slice(2,4) // starts at the third item (index of 2) and finishes at the fourth (the item with index 4 is not included)

result : ['Thor', 'Hulk']

This does not change the original avengers array. ['Captain America', 'Iron Man', 'Thor', 'Hulk', 'Hawkeye', 'Black Widow']

Using splice() removes items from an array then inserts new items in their place.

avengers.splice(3, 1, 'Scarlet Witch');

This removes the string 'Hulk' and replaces it with 'Scarlet Witch'

['Captain America', 'Iron Man', 'Thor', 'Scarlet Witch', 'Hawkeye', 'Black Widow']

This is a destructive array that changes the original array.

The splice method can also be used to insert values into an array at a specific index without removing any items - you just need to indicate that zero items are to be removed

avengers.splice(4,0,'Quicksilver');

Nothing was removed and the new value has been added into position 4

result : [ 'Captain America','Iron Man', 'Thor', 'Scarlet Witch', 'Quicksilver', 'Hawkeye', 'Black Widow' ]

When delete is used it leave the value of undefined in its place. Splice removes the value completely.

example : avengers.splice(2,1); // will remove the item at index 2 (i.e. the third item in the array) - ['Thor']

Thor has now been completely removed

['Captain America', 'Iron Man', 'Scarlet Witch', 'Quicksilver', 'Hawkeye', 'Black Widow']

reverse()

reverse() reverses the order of the array and is permanent.

avengers.reverse();

result : ['Black Widow', 'Hawkeye', 'Quicksilver', 'Scarlet Witch', 'Iron Man', 'Captain America']

sort()

This sorts the order of the array items alphabetically which is permanent

avengers.sort()

result : ['Black Widow', 'Captain America', 'Hawkeye', 'Iron Man', 'Quicksilver', 'Scarlet Witch']

Functions

Functions are chunks of code that can be referenced by a name. They help reduce repetition and allow others to follow easily.

Defining a Function

- Function Declarations

function hello() {console.log('Hello World');}

This starts with the function keyword and is followed by the name of the function, which in this case is called ' hello ', followed by parentheses. Following this is a block that contains the code for the function. This is known as a named function as the function has a name: ' hello '.

- Function Expressions

This is another way of defining a function literal. It assigns an anonymous function to a variable:

const goodbye = function() {console.log("Goodbye World!')};

They are called anonymous functions because they are not given names. We can create a named function expression instead:

const goodbye = function bye(){console.log('Goodbye World!')};

This function has the name bye and assiged to the variable goodbye.

The semicolon at the end finishes the assignment statement. A normal function does not need one as it ends in a block.

- Function() Constructors

A function can be declared using the const function() as per the example below:

const hi = new Function('console.log("Hi World!");');

This is not recommended - functions should always be declared using function literals

- Invoking a Function

A function can be invoked by running the code inside the functions body.

hello(); - functions can be invoked over and over again using this method. This will also help with reduce code repetition.

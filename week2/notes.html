Variables

They must be delared before they are used.

  1. const name = 'Alexa'; - This wont be assigned to another string
  2. let score = 0; - This value may change during the program

'typeof' can be used to check the type of variable : typeof name; (answer = 'string'

Multiple variables can be delared in the same row like this : let x = 3, y = 4, z = 5; They must be divided by commas.

Primative Data Types

  • String
  • Symbol
  • Number
  • Boolean
  • Undefined
  • Null

Scope

const amd let are blocked scope - their value belongs to the block they are declared in.

Global Scope

These are declared outside of a block. Avoid these because if they are global then they cannot be used again.

Local Scope

any variables defined within a block using const and let will only be used in that particular block.

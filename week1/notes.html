Master Mobile UX - Notes

Lesson1 : Step1 - Rewire Your Brain!

The simple guideline is whatever you are doing - do mobile first,

- Eric Schmidt (2012)

  • Mobile first design - plan mobile first or have issues later
  • Paradign shift from desktop (1 per household) to mobile devices - 2 to 3 devices per person per household
  • Rebuild to mobile first is timely and costly
  • Start small and scale up - Retrofit from big to small can be difficult

Lesson1 : Step2 - Understanding Users

  • Be there to help users through their journey
  • Be useful - make it satisfy their needs
  • Be Quick - reduce excessive steps
  • Reachability matters - make it simple for one handed users and mobile 2 handed
  • Speed Matters - not everyone has fast broadband - reduce image useage - SVG/webfonts are lower bandwidth - optimize and minify CSS and JS - GZIP files to eliminate unnecceasary data

Lesson 1 : Step 3 - Strategic Menu Design

  • We need to care abiiut our users needs
  • Move main navbar to the bottom and most used items at the bottom
  • Placement matters
  • Hamburger menu - use menu with a border instead
  • Hit users right in the happiness

Lesson 1 : Step 4 - Mobile friendly forms

  • Make it simple so people can use it easily
  • Put labels inside the fields and increse the height of the inputs
  • Use floating labels
  • Make checkboxes larger
  • Add progress bar with pagination
  • Activate autofill in forms
  • Passwords - show by default but allow users to hide if they choose
  • Use number pad for numbers not a full keyboard
  • Mobile numbers - use an anchor tag to link phone numbers to make the call

Lesson 1 : Step 5 - Mobile advertising

  • End users shape the market
  • Before you adverise, research
  • If your site has popups then you are doing it wrong
  • Don't make users have to search for the headline
  • make ads small and simple and still be effective
  • make ads small to begin with and increase in size when clicked
  • main add under main site - scrolling behind
  • STOP USING POP UPS/POP OVERS
  • Content first

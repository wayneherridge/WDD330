Transforms and Transitions - CSS3

This is another way pages can be styled without a line of Javascript

Transforms

This property translates, rotates, scales, and/or skews any element on the page. Elements can be easily manipulated.

Translation

This allows elements to be moved left, right, up, or down.

A translated element can only be moved relative to its curent position.

    
        transform: translate(45px, -45px);

This (translate(x,y)) moves an element x from the left, and y from the top.

translateX(45px) and translateY(45px) moves 45px to the right along the x axis and 45px up along the y axis.

Scaling

scaling(x,y) scales horizontally and vertically.

Rotation

rotate(10deg) rotates an element 10deg clockwise.

Skew

skew(x,y) allows elements to be skewed along the x and y axis.

Changing the Origin of the Transform

The origin of the transform can be controlled using the transform-origin property.
